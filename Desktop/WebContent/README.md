# Top Destinations - Travel Website

A beautiful travel and tourism website with booking features for flights, buses, trains, and holiday packages.

## Features

- **User Authentication**: Login and signup functionality
- **Dynamic Background**: Video background for immersive experience
- **Booking System**: 
  - Flight booking
  - Bus booking
  - Train booking
  - Holiday packages
- **Responsive Design**: Works on all devices
- **Interactive UI**: Modern and user-friendly interface

## Pages

- `index.html` - Main landing page with login/signup
- `explorreee.html` - Main exploration page
- `flight1.html`, `flight2.html` - Flight booking pages
- `bus.html`, `train.html` - Transportation booking
- `package.html`, `pack.html` - Holiday packages
- `rajasthan.html` - Destination specific page
- `payment.html` - Payment processing
- `faq.html` - Frequently asked questions

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- JavaScript (ES6+)
- Font Awesome for icons
- Responsive web design

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Explore the various booking options

## Deployment

This project is deployed using GitHub Pages. The latest version is always available at the GitHub Pages URL.
